Bootstrap 4 Reboot

Built on top of Normalize

For improved cross-browser rendering, Bootstrap uses Reboot to correct inconsistencies across browsers and devices while providing slightly more opinionated resets to common HTML elements.

Normalize.css makes browsers render all elements more consistently and in line with modern standards. It precisely targets only the styles that need normalizing.

Different from Bootstrap 3

  • Simplified Navigation
  • Some appearance font, color etc. No longer ships with glyphicons
  • Bootstrap 3 had used Normalize.css in order to render all the HTML elements their consistent appearance. Bootstrap 4 adopted an enhanced version of Normalize.css
  • Grid System — grid tier system lets the version 4 enjoy the presence of up to 5 grid tiers
  • Adopting Flexbox which does not replace the already existing page layout.
  • Bootstrap 4 offers an improved form control.
  • In the version 4, dropdown item class needs to be applied to every dropdown item.
  • Now, Bootstrap 4 demands one to properly indicate pagination and link items using page-item as well as page-link class.
  • As in Bootstrap 3, plugins were divided into two categories, namely, a file including style rules and a Javascript file. There arenâ€™t any significant changes as to how the plugins are compiled in Bootstrap 4 either. In the latest version, style rules are composed using SASS. Whereas, plugins are available under the global jQuery object as the Javascript file contains a jQuery section.
  • Bootstrap 4 offers its immense support to all the major and renowned browsers of all times, like, Safari, Chrome, Opera and Internet Explorer (10, and above) etc. If you require IE8-9 support, use Bootstrap 3.
  • In Bootstrap 4, new utility classes have been included without hampering any existing functionality, whatsoever. Such important additions are as that of responsive text alignment classes, responsive floats, and responsive embedding. Apart from offering many shortcuts, these, respectively, allow for changing the alignment of the text, floating of the elements and the scaling of the aspect ratio of any embedded media.
