    The Modern Development Stack
    MERN and MEAN an end-to-end framework for building dynamic web applications
    Mongo Express Angular React Node

    Mongo

    MongoDB : Document database – used by your back-end application to store its data as JSON (JavaScript Object Notation) documents

    Express

    Back-end web application framework running on top of Node.js.

    Angular

    Angular (formerly Angular.js): Front-end web app framework; runs your JavaScript code in the user’s browser, allowing your application UI to be dynamic

    React

    React is a JavaScript User Interface (UI) library created by Facebook for building UI components

    Node

    Node.js : JavaScript runtime environment – lets you implement your application back-end in JavaScript.
