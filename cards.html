More about cards

class .card-block adds padding all around the card.

Card titles are used by adding .card-title to a <h*> tag. In the same way, links are added and placed next to each other by adding .card-link to an <a> tag.

class .card-img-top aligns the image to the top of the card

Remove the image from the class .card-block so that it is flush. card-block provides padding.

.card-img-fluid retains image proportions.
