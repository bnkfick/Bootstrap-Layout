Bootstrap Jumbotron

Basically a way to highlight some kind of marketing message in a bannerlike fashion.

Placing the jumbotron in a container will maked it fixed width i.e. not fill 100% width of the page.

Jumbotrons have rounded corners. To change this, make the jumbotron have a class of .jumbotron-fluid

Add a container within the jumbotron to centralize the text to the middle.
